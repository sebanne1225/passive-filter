Passive Filter
BOOTH 配布パッケージ案内
version: 1.0.0

このツールは、VRChat アバターのギミックが他人視点で「出っぱなし」になるのを防ぎます。ビルド時に各ギミックの初期状態を自動で「非表示」へ倒します。

主な方針:
- このツールは非破壊で動作し、元のアバターには直接変更を加えません
- Passive Filter コンポーネントを外すだけで、元の挙動に戻せます

最初にやること:
1. GitHub repo の README を確認してください
2. まずは VCC / VPM で導入してください
3. アバターのルートに Passive Filter コンポーネントを追加してアップロードしてください

導入の基本:
- 主導線は VCC / VPM です
- Git URL 導入は補助扱いです

https://github.com/sebanne1225/passive-filter
